Задача 5

Определите булеву переменную так, чтобы, при наличии определенного блока с заданным классом, на странице она была true, иначе false. Можно использовать jQuery.

При нажатии на кнопку "Проверить" на экран выводится сообщение со значением переменной true.

Нажми на меня - и я удалюсь, а переменная поменяет свое значение на false

После удаления круга при нажатии на кнопку "Проверить" на экран выводится сообщение со значением перемнной false.

Решение

var boolean = true;

document.getElementById('block').onclick = function() {
	this.remove();
}

document.getElementById('check').onclick = function() {
	if (document.getElementById('block') == undefined) {
		boolean = false;
	}
	alert('Сейчас значение переменной ' + boolean);
}
