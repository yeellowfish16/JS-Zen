Задача 70

Написать функцию, которой на входе подаем произвольную строку и подстроку. Функция должна: обрубать пробелы в начале и конце входной строки, искать входную подстроку, возвращать true, если подстрока найдена, false – если не найдена.

Решение

document.getElementById('check').onclick = function () {
    alert(findSubstring(document.getElementById('input').value, document.getElementById('input1').value));
};

// Функция, ищущая подстроку в строке

function findSubstring(string, substring) {
    string = string.trim();

    return string.indexOf(substring) >= 0 ? true : false;
}
