Задача 11

Ноль чётный. Единица нечётная. У любого числа N чётность такая же, как у N-2. Напишите рекурсивную функцию isEven согласно этим правилам. Она должна принимать число и возвращать булевое значение. Потестируйте её на 50 и 75. Попробуйте задать ей - 1. Почему она ведёт себя таким образом? Можно ли её как-то исправить?

Решение

document.getElementById('check').onclick = function() {
	alert(isEven(document.getElementById('input').value));
};

// Проверяем четность числа
// @param strNumber - число

function isEven(strNumber) {
    var num = parseInt(strNumber);

    if (num == 0) {
        return true;
    }
    else if (num == 1) {
        return false;
    }
    else if (num < 0) {
        return isEven(-num);
    }
    else  {
        return isEven(num - 2);
    }
}
