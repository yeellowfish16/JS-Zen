Задача 6

Создайте массив arr с элементами 1, 2, 3, 4, 5 двумя различными способами. Вывести сам массив и тип данных, к которому он относится.

Решение

var arr = [1,2,3,4,5];

или

var arr = new Array(1,2,3,4,5);

document.getElementById('check').onclick = function() {
	alert('Массив ' + arr + ' имеет тип данных ' + typeof arr);
};
